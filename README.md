# ChunkBorder

###### Simple pocketmine plugin to see the chunks borders

## Config
```yaml
#     _____ _                 _    ____                _           
#    / ____| |               | |  |  _ \              | |          
#   | |    | |__  _   _ _ __ | | _| |_) | ___  _ __ __| | ___ _ __ 
#   | |    | '_ \| | | | '_ \| |/ /  _ < / _ \| '__/ _` |/ _ \ '__|
#   | |____| | | | |_| | | | |   <| |_) | (_) | | | (_| |  __/ |   
#    \_____|_| |_|\__,_|_| |_|_|\_\____/ \___/|_|  \__,_|\___|_|   
#                                                                  
#  

command_name: "chunkborder"
command_description: "Enable or disable the chunk border"
command_alias: ["border", "cb"]

border_activated: "You have activated the chunk border"
border_deactivated: "You have deactivated the chunk border"

# RGB code for the border color
rgb: "255, 0, 180"

```
